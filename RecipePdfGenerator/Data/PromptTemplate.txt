Purpose:
Use this template whenever you want to create a new recipe JSON file for the Recipe project.

Instructions:
1. Replace the placeholders in brackets [] with the actual information.
2. Keep the JSON structure consistent with Recipe.cs.

Template:

Please create a JSON recipe for my Recipe project.

Recipe URL: [Insert recipe URL here]

Recipe class structure:
- Title (string)
- SourceUrl (string)
- IngredientGroups (array of objects):
    - Header (string)
    - Items (array of strings)
- Instructions (array of strings)
- OptionalInstructions (array of strings, optional)
- ServingSuggestions (array of strings, optional)
- Customizations (array of strings, optional)

Additional instructions:
- Use the original measurements and instructions from the source.
- Convert temperatures to Celsius.
- Organize ingredients in groups if the recipe provides sections.
- For optional instructions and serving suggestions, include them in their respective fields.
- Ensure the JSON is well-formatted and ready to save as a file.
